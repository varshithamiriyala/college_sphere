# **App Name**: TechTrack

## Core Features:

- Timetable Generation: Generates 3-5 conflict-free timetables based on provided classroom, batch, subject, and faculty data, respecting all hard constraints.
- AI Constraint Suggestion Tool: Uses tool-augmented generative AI to intelligently suggest timetable rearrangements if a perfect solution is not initially feasible, aiming for minimal changes and optimal outcomes. The tool can reason over a rich database of potential changes, including trade-offs between faculty preferences, workload balancing, and room availability.
- Faculty Dashboard: Displays faculty profiles with details, working hours, leave calendar, and submission status for marks, assignments, and attendance.
- Interactive Timetable Grid: Presents a visual timetable grid with Day, Time, Room, Batch, Subject, and Faculty, allowing drag-and-drop rescheduling and conflict resolution.
- Workload Analytics: Analyzes workload distribution, leave trends, and submission compliance rates, highlighting overworked or underutilized faculty.
- Reporting: Generates downloadable PDF/CSV reports of timetables and analytics for sharing and documentation.
- User Authentication and Roles: Secure data access with role-based permissions for Admin, HOD, and Faculty.

## Style Guidelines:

- Primary color: Dark Blue (#3F51B5) to convey trust and stability, reflecting the academic environment.
- Background color: Light Gray (#F5F5F5), providing a neutral backdrop for easy readability and focus.
- Accent color: Indigo (#673AB7) to highlight key actions and interactive elements, maintaining a cohesive palette.
- Body and headline font: 'Inter', a sans-serif, providing a modern, machined, objective, neutral feel, suitable for all text.
- Use clean, line-based icons to represent actions and data categories, ensuring clarity and ease of use.
- A clean, grid-based layout will provide structured information presentation and clear hierarchy. The layout should emphasize functionality and straightforward user navigation.
- Subtle transitions and animations to provide feedback during interactions (e.g., rescheduling, report generation), enhancing user engagement without distraction.
